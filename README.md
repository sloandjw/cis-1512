# ArcadeHub
CIS-1512 Software Engineering Project<br />
<br />
Install Git: https://git-scm.com/downloads<br />
Install Visual Studio Code: https://code.visualstudio.com/ (Or IDE of your choice)<br />
<br />
Create a folder called <i>arcadehub</i> on your local computer<br />
Clone the GitHub repository in a terminal<br />
`cd arcadehub`<br />
`git clone https://github.com/sloandjw/cis-1512.git`<br />
`cd cis-1512`<br />
<br />
Pull Main<br />
`git checkout main`<br />
`git pull`<br />
<br />
Do not work on main.<br />
Do not work on main!!!<br />
Create a branch before working!<br />
`git checkout -b <your-branch>`<br />
<br />
Do Your Work<br />
In VS Code: File → Open Folder → select your local repo folder<br />
<br />
Install Live Server:<br />
Go to Extensions<br />
Search Live Server<br />
Install Live Server by Ritwick Dey<br />
(This allows you to view the website live while working on it)<br />
<br />
Commit your changes to your local repo and push your branch to GitHub<br />
`git add .`<br />
`git commit -m "Describe what you changed"`<br />
`git push -u origin <your-branch>`<br />
<br />
Then open a Pull Request on GitHub.

GitHub Pages Live Site: https://sloandjw.github.io/cis-1512/
